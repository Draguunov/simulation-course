# Отчёт по лабораторной работе

## Клеточные автоматы. Лесные пожары

### 1. Цель работы
Реализовать моделирование возникновения и распространения лесных пожаров с использованием двумерного клеточного автомата.

**Требования:**
- реализовать **не менее трёх дополнительных правил** поведения системы.


### 2. Описание модели

Модель представляет собой решётку размера `N × N`, где в программе используется `N = 50`. Каждая клетка хранит одно состояние и обновляется по правилам клеточного автомата. Обновление выполняется дискретно во времени: на каждом шаге вычисляется новая решётка `new_grid`, после чего она заменяет текущую `grid`. Такой подход гарантирует, что все переходы рассчитываются на основе одного и того же поколения.

Для каждой клетки рассматриваются 8 соседей, то есть используется окрестность Мура.

---

### 3. Состояния системы

Каждая клетка может находиться в одном из состояний:

- `EMPTY (0)` — пустая клетка.
- `ASH (1)` — пепел (остаток после горения).
- `TREE_YOUNG (2)` — молодое дерево.
- `TREE_MEDIUM (3)` — дерево среднего возраста.
- `TREE_OLD (4)` — старое дерево.
- `BURNING (5)` — горящая клетка.
- `WATER (6)` — вода (несгораемая преграда).

---

### 4. Основные правила

#### 4.1 Рост леса

Если клетка пустая, то с вероятностью `p` в ней вырастает молодое дерево:

`EMPTY -> TREE_YOUNG`

Параметр `p` задаётся пользователем слайдером `grow_slider`.

---

#### 4.2 Самовозгорание (молния)

Любое дерево может загореться без воздействия соседей с вероятностью `f`:

`TREE -> BURNING`

Параметр `f` задаётся пользователем слайдером `lightning_slider`.

---

#### 4.3 Распространение огня от соседей + влияние ветра

Если рядом есть горящая клетка, дерево может загореться. Вероятность зависит от возраста дерева и корректируется ветром.

Базовая вероятность возгорания по возрасту:
- молодое дерево — `0.2`
- дерево среднего возраста — `0.5`
- старое дерево — `0.8`

Коррекция ветром:
- если огонь находится **на наветренной стороне**, вероятность увеличивается;
- если ветер “уносит” огонь от клетки, вероятность уменьшается.

Таким образом, вероятность возгорания зависит не только от наличия горящего соседа, но и от направления ветра и возраста дерева.

---

#### 4.4 Старение деревьев

Каждое дерево может перейти в следующую возрастную категорию с вероятностью `p_age`:

`TREE_YOUNG -> TREE_MEDIUM -> TREE_OLD`

В модели используется фиксированная вероятность старения:

`p_age = 0.005`

---

#### 4.5 Пепел и восстановление почвы

После горения клетка переходит в состояние пепла:

`BURNING -> ASH`

Далее пепел со временем исчезает с вероятностью `p_clear`:

`ASH -> EMPTY`

В модели используется фиксированная вероятность очистки пепла:

`p_clear = 0.05`

---

#### 4.6 Вода как преграда

Состояние `WATER` используется как несгораемая преграда. Пользователь может рисовать воду мышью: вода блокирует распространение пожара и не меняется по правилам автомата.

Интерактивное управление:
- **ЛКМ** — поджечь дерево (перевести дерево в `BURNING`)
- **ПКМ** — поставить воду (`WATER`)

---

### 5. Графический интерфейс (GUI)

GUI реализован с использованием `tkinter`.

Интерфейс включает:
- кнопку **Старт/Стоп** для запуска и остановки симуляции;
- кнопку **Очистить поле** для сброса модели;
- слайдер **p** для управления ростом леса;
- слайдер **f** для управления частотой самовозгорания;
- выбор направления ветра с помощью радиокнопок;
- легенду состояний слева;
- интерактивное управление мышью.

Обновление модели выполняется каждые `100 мс`.

---

### 6. Результаты моделирования

В ходе экспериментов наблюдаются характерные эффекты:

- При увеличении `f` возрастает число очагов возгорания.
- При увеличении `p` лес быстрее восстанавливается после пожара.
- При включении ветра фронт огня становится направленным: огонь быстрее распространяется по ветру.
- Старые деревья выгорают быстрее, чем молодые.
- Вода эффективно останавливает распространение огня и формирует “границы” пожара.
- Пепел замедляет восстановление выгоревших участков и делает модель более реалистичной.

### 7. Выводы

В ходе работы реализован двумерный клеточный автомат для моделирования лесных пожаров с интерактивным GUI. Базовые процессы — рост леса, самовозгорание и распространение огня — дополнены правилами, повышающими реализм модели: влияние ветра, старение деревьев, различная горючесть деревьев по возрасту, стадия пепла и водные преграды.

Разработанная модель позволяет наблюдать типичную динамику лесного пожара: появление очагов возгорания, расширение фронта огня, образование выгоревших зон и последующее восстановление леса.
