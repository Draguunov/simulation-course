### Базовый датчик случайных чисел

**Задание:**
- реализовать базовый датчик случайных чисел;
- вычислить выборочные среднее и дисперсию:
  - для реализованного датчика;
  - для встроенного генератора языка программирования;
- размер выборки — **100 000 значений**;
- сравнить результаты с теоретическими;
- сделать вывод.

## Выводы

В ходе выполнения лабораторной работы был реализован **мультипликативный конгруэнтный генератор (MCG)** — базовый датчик случайных чисел, соответствующий методу, указанному в презентации.  
Генератор построен на рекуррентном соотношении  

$$X_{n+1} = (a \cdot X_n) \bmod m$$

с параметрами $a = 16807$, $m = 2^{31} - 1$ (минимальный стандарт Park–Miller). Полученные псевдослучайные числа приводятся к интервалу $[0, 1)$ делением текущего состояния на $m$.

Для оценки качества генерации были вычислены выборочное среднее и выборочная дисперсия (смещённая оценка) на выборке объёмом $N = 100\,000$.  
Сравнение проводилось с теоретическими значениями для равномерного распределения $U[0,1]$ ($\mu = 0.5$, $\sigma^2 = 1/12 \approx 0.0833333$) и с результатами встроенного генератора Python `random.random()`, реализующего алгоритм Mersenne Twister.

**Результаты:**

| Характеристика | Теоретическое значение | MCG (Park–Miller) | Встроенный генератор (MT) |
|----------------|------------------------|-------------------|---------------------------|
| Среднее        | 0.500000               | 0.499986          | 0.499921                  |
| Дисперсия      | 0.0833333              | 0.083313          | 0.083296                  |
| Погрешность среднего | —                | 1.4×10⁻⁵          | 7.9×10⁻⁵                  |
| Погрешность дисперсии | —                | 2.0×10⁻⁵          | 3.7×10⁻⁵                  |

**Анализ:**

1. **Соответствие методическим указаниям** – реализованный датчик полностью соответствует заданию: использован **мультипликативный конгруэнтный метод** без аддитивной константы, что отличает его от линейных конгруэнтных генераторов (LCG), применённых в некоторых студенческих работах.

2. **Статистические свойства** – отклонения выборочного среднего и дисперсии от теоретических значений не превышают $10^{-4}$, что подтверждает равномерность распределения и отсутствие систематических смещений.

3. **Сравнение с встроенным генератором** – оба датчика демонстрируют близкие результаты. Небольшое преимущество по точности в данном эксперименте оказалось у реализованного MCG, однако это может быть следствием статистической вариации.

4. **Требования к базовому датчику** – разработанный генератор удовлетворяет всем необходимым критериям:
   - **Математические**: равномерное распределение, отсутствие корреляции, апериодичность (период $m-1 \approx 2.1 \times 10^9$), воспроизводимость при фиксированном seed.
   - **Вычислительные**: быстродействие (сложность $O(1)$), портируемость, малый объём памяти.

Таким образом, мультипликативный конгруэнтный генератор с параметрами Park–Miller является корректной и эффективной реализацией базового датчика случайных чисел, пригодной для использования в имитационном моделировании.

По памяти генератор является экономичным, так как для его работы достаточно хранить только текущее состояние и параметры метода.

При этом встроенный генератор Python остаётся более универсальным средством, обладающим лучшими статистическими характеристиками для более сложных практических применений.
