## Теоретические сведения

### 1. Дискретная случайная величина

Пусть дискретная случайная величина задана рядом распределения:

$\[
X = \{x_1, x_2, ..., x_m\}
\$] 

с вероятностями

$\[
P = \{p_1, p_2, ..., p_m\}, \quad \sum_{i=1}^{m} p_i = 1
\$]

#### Теоретическое математическое ожидание:
$\[
M(X) = \sum_{i=1}^{m} x_i p_i
\$]

#### Теоретическая дисперсия:
$\[
D(X) = \sum_{i=1}^{m} x_i^2 p_i - M(X)^2
\$]

#### Эмпирическая вероятность:
$\[
\hat p_i = \frac{n_i}{N}
\$]

где:
- `n_i` — число появлений значения `x_i`,
- `N` — объём выборки.

#### Выборочное среднее:
$\[
\bar x = \frac{1}{N} \sum_{i=1}^{N} x_i
\$]

#### Выборочная дисперсия:
$\[
D^* = \frac{1}{N} \sum_{i=1}^{N}(x_i - \bar x)^2
\$]

#### Относительная погрешность:
$\[
\varepsilon = \frac{|x_{\text{эмп}} - x_{\text{теор}}|}{|x_{\text{теор}}|} \cdot 100\%
\$]

---

### 2. Критерий χ²

Для проверки гипотезы о согласии эмпирического распределения с теоретическим используется статистика:

$\[
\chi^2 = \sum \frac{(n_i - np_i)^2}{np_i}
\$]

где:
- `n_i` — наблюдаемая частота;
- `np_i` — ожидаемая частота.

Если

$\[
\chi^2_{\text{набл}} < \chi^2_{\text{кр}}
\$]

то гипотеза о соответствии эмпирического распределения теоретическому **не отвергается**.

Если

$\[
\chi^2_{\text{набл}} > \chi^2_{\text{кр}}
\$]

то гипотеза **отвергается**.

---

### 3. Нормальное распределение

Нормальная случайная величина с параметрами `a` и `σ²` имеет плотность:

$\[
f(x) = \frac{1}{\sigma \sqrt{2\pi}} e^{-\frac{(x-a)^2}{2\sigma^2}}
\$]

В программе используется метод Бокса–Мюллера для генерации стандартной нормальной случайной величины:

$\[
Z = \sqrt{-2 \ln U_1} \cdot \cos(2\pi U_2)
\$]

где `U1` и `U2` — независимые случайные величины, равномерно распределённые на интервале `[0,1)`.

Затем выполняется преобразование:

$\[
X = a + \sigma Z
\$]

## Алгоритм работы программы

### Генерация дискретной случайной величины
1. Пользователь вводит значения `x_i` и вероятности `p_i`.
2. Генерируется случайное число `α` из интервала `[0,1)`.
3. Из `α` поочерёдно вычитаются вероятности `p_i`.
4. Когда `α <= 0`, выбирается соответствующее значение `x_i`.
5. Процедура повторяется `N` раз.
6. Далее вычисляются статистические характеристики.

### Генерация нормальной случайной величины
1. Пользователь задаёт `a`, `σ²` и `N`.
2. Генерируются массивы равномерных случайных чисел `U1` и `U2`.
3. Применяется формула Бокса–Мюллера.
4. Полученные значения преобразуются к распределению `N(a, σ²)`.
5. По выборке строится гистограмма и вычисляются статистические характеристики.

### 3. Вероятностные модели и алгоритмы

#### 3.1. Моделирование дискретной СВ
Используется **метод последовательного вычитания** (эквивалент метода обратной функции).  
Отрезок $[0,1]$ разбивается на подотрезки длиной $p_i$. Генерируется $\alpha \sim U(0,1)$. Последовательно вычитаются вероятности:
- если $\alpha \le p_1$, выбирается $x_1$;
- иначе $\alpha := \alpha - p_1$, и проверка $\alpha \le p_2$, и т.д.

#### 3.2. Моделирование нормальной СВ (метод Бокса–Мюллера)
Для генерации стандартной нормальной величины $z$ используются две независимые равномерные величины $u_1, u_2 \in (0,1]$:
$$
z = \sqrt{-2\ln u_1} \cdot \cos(2\pi u_2)
$$
Итоговая выборка: $x = \mu + \sigma \cdot z$, где $\sigma = \sqrt{\sigma^2}$.

#### 3.3. Статистические показатели
- **Выборочное среднее:** $\displaystyle \bar{x} = \frac{1}{N}\sum_{i=1}^{N} x_i$
- **Выборочная дисперсия (смещённая):** $\displaystyle S^2 = \frac{1}{N}\sum_{i=1}^{N} (x_i - \bar{x})^2$
- **Относительная погрешность:** $\displaystyle \delta = \frac{|\theta_{th} - \theta_{emp}|}{|\theta_{th}|}\cdot 100\%$
- **Критерий $\chi^2$ Пирсона (для ДСВ):**
  $$
  \chi^2_{\text{stat}} = \sum_{i=1}^{m} \frac{(n_i - N p_i)^2}{N p_i},
  $$
  где $n_i$ – наблюдаемая частота значения $x_i$, $m$ – число значений.
  Степени свободы: $df = m-1$. Гипотеза о соответствии распределения принимается при $\chi^2_{\text{stat}} < \chi^2_{\text{крит}}(0.95, df)$.
